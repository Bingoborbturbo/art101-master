Creating a project in Blender (Task 1)

  1. open the Blender application
  2. Exit the splash menu to create a new file, or select an existing one
  3. Append any previous blend files or import any meshes or textures using the dropdown menu
  4. Using basic meshes, block out your scene
  5. Import and position a camera for your scene
  6. Import and position lighting
  7. From your blocked out scene, model and texture any objects, buildings, people, etc. in your program of choice
  8. Replace blocked out models with high quality meshes and textures for each object
  9. Tweak render settings in the viewport before rendering the final scene
  10. Render final scene, and apply any post-processing effects in a separate image editing program
